# Gerbers
This folder contains manufacturing outputs for the FreeRunner PCB.

## Contents
- Gerbers
- BOM
- Pick and Place (Position) files

Use the pick and place files if you intend on having a manufacturer make the PCBs for you. If you are having these PCBs assembled, please pay attention to the orientation/placement of J2 as it may be slightly off.
