# Gerbers
This folder contains manufacturing outputs for the FreeRunner PCB.

## Contents
- Gerbers
- BOM
- Pick and Place (Position) files

Use the pick and place files if you intend on having a manufacturer make the PCBs for you. **These exact files have been used to order from JLCPCB, the PCBs turned out amazing and worked perfectly.**
